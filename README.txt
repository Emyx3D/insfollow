## Requirements

1. openssl
2. curl

## How to Install in Termux

$ pkg up -y
$ pkg install openssl-tool
$ pkg install curl
$ cd insfollow
$ chmod +x insfollow.sh
$ termux-wake-lock
$ bash insfollow.sh



## How to Install on Linux

$ apt update 
$ apt install openssl-tool
$ apt install curl
$ cd insfollow
$ chmod +x insfollow.sh
$ ./insfollow.sh
